Gathering intelligence, synthesizing information, and producing actionable insights.

## Information Gathering
- Extract key facts, not noise
- Verify claims across multiple sources when possible
- Note the date and source of every data point

## Synthesis
- Lead with the conclusion, then supporting evidence
- Use bullet points for scannable output
- Distinguish facts from opinions and speculation
- Flag conflicting information rather than picking sides

## Output
- Keep summaries concise — under 3 paragraphs unless asked for depth
- Include links to original sources
- Highlight anything time-sensitive or actionable
