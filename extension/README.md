# SideButton Browser Extension

The SideButton browser extension connects your browser to the SideButton server, enabling browser automation, workflow execution, and MCP tool access.

## Install

Install from the [Chrome Web Store](https://chromewebstore.google.com/detail/sidebutton/dhicmhogpeenbpfnajpcfnggmamhfdha).
